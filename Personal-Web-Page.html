Publications

Currently I have two publications, both from my time working at Vulcan, which are in the submission process to journals:

  • Sub-10-attosecond Fibre-length Stabilisation using Phase-locked Spectral Interferometry. This was the main body of work undertaken whilst on placement at Vulcan. Path length synchronisation is important for next generation laser facilities for instance, where the delay between pulses wants to be controlled to the femtosecond regime. We investigated path length stabilisation using phase-locked spectral interferometry, and achieved stabilisation results sub-10 attoseconds.
